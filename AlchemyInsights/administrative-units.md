# <a name="administrative-units"></a>Beheereenheden

Een beheereenheid is een Azure AD-resource die een container kan zijn voor andere Azure AD-bronnen. Een beheereenheid kan alleen gebruikers en groepen bevatten.

Zie Beheereenheden in Azure Active Directory voor meer informatie [over beheereenheden.](/azure/active-directory/roles/administrative-units)
