# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Een gedeeld postvak gebruiken op een mobiel apparaat

Als u een gedeeld postvak op uw [](https://products.office.com/outlook-mobile-for-android-and-ios) mobiele apparaat wilt gebruiken, moet u Outlook mobiel downloaden en vervolgens een gedeeld postvak toevoegen [aan Outlook mobiele telefoon.](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f)
  
Hulp krijgen: Help in de app voor Outlook [voor iOS en Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
